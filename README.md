# ProLineups Optimizer: AI-Driven Fantasy Cricket Engine

## Overview
A robust Python backend designed to generate mathematically optimized fantasy cricket lineups for high-stakes IPL tournaments. This system processes player statistics and match variables to construct high-probability squads, removing human bias from lineup creation.

## Core Features
* **Algorithmic Lineup Generation:** Automatically balances player credits, team constraints, and positional roles to output tournament-ready Dream11 squads.
* **Data-Driven Strategy:** Evaluates historical performance and match-ups to identify optimal captain and vice-captain selections.
* **Scalable Python Backend:** Clean, modular architecture ready for further predictive modeling integration.
